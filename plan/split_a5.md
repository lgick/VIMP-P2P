# A5. Разделение документации ✅важно ✅ выполнен

Две документации: движковая (остаётся в репо движка) и игровая (переезжает в репо
игры). Разметка по инвентарю:

| Страница | Куда |
| --- | --- |
| `master.md`, `host.md`, `plugin-api.md`, `deployment.md`, `client.md`, `network.md` | **движок** |
| `gameplay.md`, `extending.md` | **игра** |
| `configuration.md`, `core.md`, `architecture.md`, `getting-started.md` | **распилить** на движковую и игровую половины (заголовки уже разделяют разделы) |

- Каждый репо держит свои `docs/en` + `docs/ru` (правило зеркалирования из
  CLAUDE.md сохраняется в обоих).
- `plugin-api.md` — движковый, но это шов, на который ссылается игра; в игровых
  доках дать ссылку на опубликованный контракт.
- Обновить корневые `README.md` обоих репозиториев и таблицу «Change → page» в
  их `CLAUDE.md`.
- SDK для сторонних разработчиков — **вне рамок** (задел на будущее, как указал
  пользователь), но структура плагин-контракта под него уже готова.

## Критические файлы

`docs/**` (распил).

## Итог

При выполнении обнаружилось, что `docs/**` в этом репозитории не были
синхронизированы с уже случившимся физическим переносом игры в `vimp-tanks`
(A3): все страницы ссылались на пути `games/tanks/...`, которых в дереве
репозитория больше нет. Разошлось шире исходной таблицы — распилены и
исправлены все страницы, не только помеченные MIXED:

- **Движок** (`vimp-engine/docs/en+ru/`): `architecture.md`,
  `configuration.md`, `core.md`, `getting-started.md` — обрезаны до
  движковой половины (WASM-сборка, `core:build*`, `game:build`,
  `maps:export` реально живут только в `vimp-tanks/package.json`, здесь их
  больше нет — документация теперь соответствует фактическим
  npm-скриптам). `gameplay.md` и `extending.md` удалены отсюда полностью.
  Во всех оставшихся страницах (`master.md`, `host.md`, `client.md`,
  `network.md`, `auth.md`, `plugin-api.md`, `README.md`) заменены
  устаревшие пути `games/tanks/...` на формулировки «игра-плагин
  (например, `vimp-tanks`)» или ссылки на GitHub того репозитория;
  навигационные футеры (Prev/Next) перелинкованы без выпавших страниц.
- **Игра** (`vimp-tanks/docs/en+ru/`, репозиторий не имел `docs/` и
  `CLAUDE.md` вообще): создан с нуля — `README.md` (оглавление),
  `architecture.md`, `configuration.md`, `core.md`, `getting-started.md`
  (игровые половины, пути `games/tanks/src/...` → `src/...`), плюс
  перенесённые `gameplay.md`/`extending.md`. Корневые `README.md` и
  `CLAUDE.md` (компактный, на английском, с таблицей Change → page)
  созданы в `vimp-tanks`.
- Битых относительных ссылок не осталось (проверено grep'ом по обоим
  деревьям `docs/`); кросс-репо ссылки — абсолютные GitHub URL
  (`blob/main/docs/{en,ru}/...`).
- Не сделано: живая проверка рендеринга ссылок на github.com (репозитории
  ещё не запушены/не публичны в момент выполнения) — только текстовая
  сверка путей.
